🤖 AI Chatbot (Gemini + Flask)

A web-based AI chatbot powered by Google Gemini API, built with Flask, and deployed online using Render.
The chatbot supports conversation memory, chat history, markdown responses, and a responsive chat interface.

🌐 Live Demo:
https://my-ai-chatbot-1-n3w0.onrender.com

🚀 Features

💬 AI Chat Interface – Ask questions and receive intelligent responses from Gemini AI.

🧠 Conversation Memory – The chatbot remembers previous messages to maintain context.

🗂 Chat History Storage – All messages are stored using SQLite.

🔄 Chat History API – Retrieve previous conversation messages.

🧹 Clear Chat Option – Reset the conversation anytime.

✨ Typing Indicator – Shows when the AI is generating a response.

🧾 Markdown Rendering – AI responses support formatted text and code blocks.

📱 Responsive UI – Works on both desktop and mobile devices.

🔐 Secure API Keys – Environment variables used for API key protection.

☁️ Cloud Deployment – Hosted on Render.
